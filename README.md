# DWV (Do With Voice)

DWV is a voice-driven system that performs real-world actions using spoken commands.

---

## Tools Used

* [SoundDevice](https://python-sounddevice.readthedocs.io/en/0.5.3) — Capture audio
* [Whisper](https://github.com/openai/whisper) — Speech-to-Text (STT)
* [OpenClaw](https://openclaw.ai/) — agent execution
* [Coqui TTS](https://github.com/idiap/coqui-ai-TTS) — Text-to-Speech (TTS)
* [OpenWakeWord](https://openwakeword.com/) — Wake word detection

---
